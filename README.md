## Features

| Feature | Description |
|--------|-------------|
| **AI-Powered Chat** | Uses **Gemini 2.0 Flash** for fast, accurate, and context-aware responses |
| **Material You 3 Design** | Beautiful, expressive UI with dynamic color, elevation, and ripple effects |
| **Dark & Light Mode** | Auto-detects system preference with smooth transitions |
| **Persistent Memory** | Chat history saved in `localStorage` — never lose context |
| **Markdown Support** | Renders **bold**, *italics*, `code`, lists, and headers beautifully |
| **Mobile-First & Responsive** | Works perfectly on phones, tablets, and desktops |
| **PWA Ready** | Installable as a web app with manifest and offline feel |
| **API Key Rotation** | Supports **18+ keys** for **unlimited quota** (client-side rotation) |
| **Zero Dependencies** | No React, no Vue, no Node — just vanilla code |
